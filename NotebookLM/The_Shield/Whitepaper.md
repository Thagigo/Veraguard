# VeraGuard Whitepaper (v2.1)

## Abstract
VeraGuard is a decentralized security protocol utilizing "Ghost Agents" to perform continuous forensic analysis of smart contracts.

## Core Components
1. **The Shield**: Knowledge base of known exploit vectors.
2. **The Vault**: User history and sovereign certificates.
3. **The Ledger**: Transparent accounting of all protocol fees.

## Tokenomics
- 60% Security Vault (Insurance)
- 25% Sheriff Yield (Stakers)
- 15% War Chest (Bounties)
